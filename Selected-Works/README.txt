Drop your project's MAIN image here (one per project), then add a
matching entry in  ../data/works.js  with the exact filename.

Tips:
- Use web-friendly sizes: roughly 1600px wide, JPG, saved at ~80% quality.
  Huge images make the site slow.
- Filenames: lowercase, hyphens-instead-of-spaces (e.g. light-shield.jpg),
  because the live server is case-sensitive.
